Ngx-Google-Fonts

This is a pure-Angular library which provdes an input component for a user to select a web safe font or google font. It includes:

  • Loading the list of Google fonts.
  • Proving a select-like interface for selecting a font.
  • Lazy-loading fonts so the select can render the fonts as the user scrolls.
  • Loading the currently selected font, if a font is initially provided to the component.
  • Prefix-based font lookup.

To test this out, get a Google API key, enter it into the textarea below, and click "Load"



The currently selected font is {{ selectedFont$ | async }}, and the font the mouse is hovering over is {{ hoverFont$ | async }}.

We can use this paragraph to ensure the selection overlay does not shift the elements under it.
